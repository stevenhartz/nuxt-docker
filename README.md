# Nuxt Docker
Dockerise Nuxt.js App

## Run docker
```bash
make up
```

## Stop docker
```bash
make down
```

## First installation for initialize the project
```bash
./install.sh
```

### Install the dependencies
```bash
make yi
```

### Update the dependencies
```bash
make yu
```

### Makefile Help (show all commands)
```bash
make help
```

## Development Server

Start the development server on http://localhost:3000
